# 🌙 꿈해몽 & 운세

AI 기반 꿈 해석과 오늘의 운세, 타로 한 장까지! 당신의 꿈과 오늘을 해석해드립니다.

## 주요 기능

### 🌌 꿈해몽
- 꿈에서 본 것을 키워드로 입력
- 40개 이상의 꿈 해석 데이터베이스
- 행운지수 표시
- 결과 공유 기능

### ⭐ 오늘의 운세
- 12별자리 선택
- 연애운/금전운/직장운/행운 별점
- 매일 바뀌는 운세 메시지
- 행운의 색상/숫자 추천

### 🃏 타로 한 장
- 3장 중 1장 선택
- 15개 메이저 아르카나 카드
- 카드 의미와 조언 제공

## 기술 스택

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Type:** Progressive Web App (PWA)
- **Style:** 2026 Glassmorphism, Dark Mode First
- **Theme:** 신비/몽환 (#9b59b6)
